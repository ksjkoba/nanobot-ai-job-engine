# NanobotAI: Autonomous Job Application Engine

An AI-powered automation suite designed to scrape job boards (LinkedIn, Naukri), analyze JD requirements, and programmatically submit tailored applications.

## Key Features
* **Web Automation:** Built with Python and Playwright for high-speed, headless browser interaction.
* **LLM Integration:** Utilizes Gemini/OpenAI API to dynamically tailor resumes and cover letters based on specific Job Descriptions.
* **Smart Filtering:** Automated logic to bypass irrelevant roles and focus on IAM/IGA positions.
* **Local Processing:** Generates `Tailored_Application.pdf` locally before submission.

## Tech Stack
* Python 3.x
* Playwright (Chromium)
* LLM (Gemini API)
